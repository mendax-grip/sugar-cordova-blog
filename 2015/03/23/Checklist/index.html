      CheckList for Sugar-Cordova Project Application

      The pre-project for application for Sugar-Cordava Application Project is a list of around 8 requirements which Puneet Kaur had put in her previous year GSoC blog. These were definitive for someone like me who started late applying for GSoC ‘15 and entering into the world of open-source program development.

      So, I am going to list out those requirements right here, so that I can come back and cross them-out as I progress. I must add here, my psuedo-compulsive habit of documenting the progress has been effective in every pursuit I have taken, so it’s quite natural for me to take pride in documenting my failures and success. It helps me to focus on the real goals and avoid the distraction. It’s actually priviledge rather a burden to document.

      The List

      • Go through all the post on this blog

      • Build activities using Sugar-Web and connect using python-javascript bridge as suggested in the post.

      • Go through my sugar cordova related repos on github : Puneet Kaur

      • Setup Cordova for Sugar with the help of my repos : Cordova-cli, Cordova-Lib and Cordova-Plugman.

        • Clone these repos, follow the README on each repo and install Cordova for Sugar on your system.

        • Make a simple web app for Sugar with the help of this installation by using simple commands:

        • You’ll get an XO which you must try installing on sugar development environment using sugar-install-bundle command.

      • Setup your Sugar development environment and explore around try making changes in code, see where the code rests and try making a few changes to the code, play around with the GUI and get a feel of the sugar environment if you aren’t familiar before.

      • Know all about cordova - from in and out , visit the cordova repos : Cordova-lib,Cordova-cli, Cordova-Plugman, Cordova-js and some platforms repo too: Cordova-Android, Cordova-IOS etc.

      • Read the docs : Cordova-Docs

      • Know about sugar web architecture: Web Architecture, Sugar-Web and GSoc ‘13 blog

      • Carefully read through the last post of how the Cordova plugins are made

        • Make a demo plugin for Sugar

      • Suggest a list of plugins you aim to code this summer along with the relevant Workflow as to how you think you’ll approach each of them

      • Lastly, keep a Blog about your progress on these points, a record of all what you have done and whats remaining.

      Let’s hope I complete all the requirments before the deadline.
